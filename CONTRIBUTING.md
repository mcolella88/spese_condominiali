# Contributing

Grazie per il tuo interesse a contribuire a Gestione Spese Condominiali!

## Come Contribuire

1. Fai un fork del progetto
2. Crea un branch per la tua feature (`git checkout -b feature/AmazingFeature`)
3. Commit le tue modifiche (`git commit -m 'Add some AmazingFeature'`)
4. Push al branch (`git push origin feature/AmazingFeature`)
5. Apri una Pull Request

## Suggerimenti per lo Sviluppo

- Mantieni il codice semplice e ben commentato
- Segui lo stile esistente del codice
- Testa le modifiche su più browser quando possibile
- Aggiorna la documentazione se necessario

## Segnalazione Bug

Per segnalare bug, apri una issue su GitLab con:
- Descrizione del problema
- Passaggi per riprodurlo
- Browser e versione utilizzati
- Screenshot se utile
